PROBLEMA 5: Sem proteção de segurança

  ✅ DEPOIS: X-Content-Type-Options, X-Frame-Options, etc

TROUBLESHOOTING RÁPIDO
================================================================================

Erro: 404 em /listas
  Causa: Build não foi feito corretamente
  Solução:
    1. Verificar BACKEND_URL está configurada
    2. Limpar cache: Settings > Advanced > Clear Build Cache
    3. Redeploy

Erro: 502 Bad Gateway em /api/*
  Causa: BACKEND_URL incorreta ou backend offline
  Solução:
    1. Verificar BACKEND_URL em Environment Variables
    2. Testar se backend está online
    3. Redeploy

Erro: Cannot find module
  Causa: Case sensitivity (Windows vs Linux)
  Solução:
    1. Procurar imports errados
    2. Corrigir: ./component → ./Component
    3. Commit & push

Erro: Build failed
  Causa: Dependências ou erro de sintaxe
  Solução:
    1. Rodar build local: npm run build
    2. Verificar erros
    3. Corrigir e fazer commit
    4. Push
